# Pankaj Kumar - Professional Portfolio

A modern, high-performance portfolio website built with React 19, Tailwind CSS v4, and Framer Motion. This project showcases my career evolution, technical ecosystem, and professional certifications with a sleek, interactive, and dark-mode-ready interface.

## 🚀 Features

- **React 19:** Leveraging the latest features of React for optimal performance.
- **Tailwind CSS v4:** Using the cutting-edge utility-first CSS framework for rapid styling.
- **Dark Mode:** Seamlessly switch between light and dark themes with persistent user preference.
- **Micro-interactions:** Smooth animations and hover effects using Framer Motion (via `motion/react`).
- **Responsive Design:** Fully optimized for mobile, tablet, and desktop devices.
- **Lucide Icons:** Beautiful, consistent iconography throughout the application.

## 🛠️ Tech Stack

- **Frontend:** React, TypeScript, Vite
- **Styling:** Tailwind CSS v4
- **Animations:** Framer Motion (`motion`)
- **Icons:** Lucide React

## 📦 Getting Started

Follow these instructions to get a copy of the project up and running on your local machine.

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) (v18 or higher) and `npm` installed.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/pankaj2k16/portfolio.git
   cd portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```
   Open [http://localhost:5173](http://localhost:5173) in your browser to see the result.

### Building for Production

To create an optimized production build, run:
```bash
npm run build
```
The output will be in the `dist` folder.
